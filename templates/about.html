A test app combining rust's Actix-web framework, sqlite database, and tera templating engine to serve both server-side rendered templates and client-side VueJS page content.

Query parameters, json data, and route url parameters can be mapped to the async function views which respond to them, in most cases directly into structs that validate types. Json data can be received and returned as typed objects, for example app's User model.

The project loosely follows Django conventions for urls, views, models, etc. contained in one or more app folders under the root source directory. The main.rs launches the server processes and calls on modules to add in all accessible url routes and handlers.
